### **¬øQu√© es NODE_ENV?**

`NODE_ENV` es una variable de entorno que define el **modo de ejecuci√≥n** de tu aplicaci√≥n Node.js.

|---------|---------------|--------------|
| **Logs** | Detallados (f√°cil debug) | M√≠nimos (solo errores) |
| **Stack Traces** | Completos | Ocultos |
| **Cache** | Deshabilitado | Habilitado |
| **Optimizaci√≥n** | Baja | Alta |
| **Velocidad** | Lenta | R√°pida |
| **Seguridad** | Relajada | Estricta |

